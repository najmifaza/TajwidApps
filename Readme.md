# TajwidDigital - Platform Belajar Tajwid #1

TajwidDigital adalah platform edukasi interaktif yang dirancang untuk membantu umat Muslim mempelajari hukum bacaan Al-Qur'an (Tajwid), makharijul huruf, dan sifat huruf secara fasih dan tartil. Template ini dibangun menggunakan framework Bootstrap v5.3.3 untuk memastikan tampilan yang responsif dan modern.

* **Materi Lengkap**: Tersedia menu dropdown materi yang mencakup berbagai kategori Tajwid.
* **Desain Responsif**: Platform dapat diakses dengan optimal melalui perangkat desktop, tablet, maupun ponsel.
* **Statistik Pengguna**: Menampilkan data pencapaian seperti jumlah klien, proyek, dan jam dukungan.
* **Testimonial & FAQ**: Menyediakan ulasan pengguna dan jawaban atas pertanyaan umum seputar pembelajaran tajwid.
